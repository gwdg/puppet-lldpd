Puppet Class: lldpd

Inherits:
::lldpd::params
Defined in:
manifests/init.pp

Overview

Installs and manages the LLDP agent.

Examples:

Declaring the class

include ::lldpd

Enabling CDPv1 and CDPv2

class { '::lldpd':
  enable_cdpv1 => true,
  enable_cdpv2 => true,
}

Enable SNMP support

class { '::lldpd':
  enable_snmp => true,
  snmp_socket => ['127.0.0.1', 705],
}

Parameters:

  • addresses (Optional[Array[String, 1]]) (defaults to: undef) —

    List of IP addresses to advertise as the management addresses.

  • chassis_id (Optional[Array[String, 1]]) (defaults to: undef) —

    List of interfaces to use for choosing the MAC address used as the chassis ID.

  • class (Optional[Integer[1, 4]]) (defaults to: undef) —

    Set the LLDP-MED class.

  • enable_cdpv1 (Variant[Boolean, Enum['force']]) (defaults to: false) —

    Disable, enable, or force Cisco Discovery Protocol version 1. Both this and the below parameter are combined to form an overall CDP setting, not all combinations are supported by lldpd.

  • enable_cdpv2 (Variant[Boolean, Enum['force']]) (defaults to: false) —

    Disable, enable, or force Cisco Discovery Protocol version 2. Both this and the above parameter are combined to form an overall CDP setting, not all combinations are supported by lldpd.

  • enable_edp (Variant[Boolean, Enum['force']]) (defaults to: false) —

    Disable, enable, or force the Extreme Discovery Protocol.

  • enable_fdp (Variant[Boolean, Enum['force']]) (defaults to: false) —

    Disable, enable, or force the Foundry Discovery Protocol.

  • enable_lldp (Variant[Boolean, Enum['force']]) (defaults to: true) —

    Disable, enable, or force the Link Layer Discovery Protocol.

  • enable_sonmp (Variant[Boolean, Enum['force']]) (defaults to: false) —

    Disable, enable, or force the Nortel Discovery Protocol.

  • enable_snmp (Boolean) (defaults to: false) —

    Enable or disable the SNMP AgentX sub-agent support.

  • interfaces (Optional[Array[String, 1]]) (defaults to: undef) —

    List of interfaces to advertise on.

  • package_name (String) (defaults to: $::lldpd::params::package_name) —

    The name of the package.

  • service_name (String) (defaults to: $::lldpd::params::service_name) —

    Name of the service.

  • snmp_socket (Optional[Variant[Stdlib::Absolutepath, Tuple[IP::Address::NoSubnet, Bodgitlib::Port]]]) (defaults to: undef) —

    The path or IP & port pair used to establish an SNMP AgentX connection.
